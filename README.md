# Development Env

For bash and vim environment yet. 

## Usage

After clone this repository, you should put a follwing line in `~/.barch` file. 

```bash
source ~/env_setting/bashrc
```

And you exectue init scripts

```bash 
~/env_setting/init.sh
````
